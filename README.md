学生记账本 (Student Ledger)
学生记账本是一款旨在帮助学生群体树立健康消费观的精细化财务管理应用。系统围绕学生的“月度生活费”核心场景展开，提供直观的额度预警、日均开销计算以及全方位的可视化收支统计。
核心功能模块
智能首页看板 (Home Dashboard)

多维度预算追踪：实时呈现本月余额、本月支出与本月收入。

消费速率预警：根据剩余天数自动计算日均消费与本月剩余日均可用额度，防止超支。

发放周期提醒：动态展示距发钱日倒计时（如临近的 10天），精准契合学生党的生活费发放节奏。

便捷账单明细 (Ledger Details)

流式账单记录：按时间轴顺序展现消费明细，支持餐饮、其他等多分类标签。

快速检索过滤：内置分类与备注搜索框，支持全部、支出、收入的一键切换过滤。

可视化数据统计 (Data Statistics)

周期趋势分析：提供近6个月趋势柱状图，直观掌控长期消费走向。

消费结构异动：通过高可读性的进度条展示支出分类占比（如 94%、5% 等具体权重），轻松抓取大额开支。

系统与数据管理 (Settings & Data)

视觉体验优化：全界面深度适配主题模式中的深色模式，保护视力且兼具科技感。

数据主权掌控：支持通过 JSON 文件进行导出数据备份与导入数据恢复，保障用户数据不丢失。

灵活预算定制：允许自定义每月生活费基准额度（如 30000 元）及具体的发放日期（如 1号）。
